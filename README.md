# 🥕 Slime Legion Kids: Professional Gaming Collection
**Status:** PROD_READY | **Deploy:** GitHub Pages | **Vibe:** High-Fidelity Kids Gaming

Esta colección reúne las herramientas lúdico-educativas de la **Slime Legion**, diseñadas bajo estándares de alta fidelidad para el desarrollo infantil, atención conjunta y aprendizaje sensorial.

## 🎮 Juegos Incluidos

### 1. [Mini Zanahoria: La Aventura Adaptativa](./mini-zanahoria/)
Un módulo diseñado para trabajar la **atención conjunta**, la **resonancia emocional** y la estructuración de **rutinas diarias**.
- **Mecánicas:** Focus Orbit, Emotion Resonance, Routine Builder.
- **Tecnología:** Vanilla JS + CSS Grid (Aesthetic Robbit Palette).

### 2. [Abecedario Mágico: Caligrafía Terapéutica](./magic-alphabet/)
Herramienta terapéutica de caligrafía y motricidad fina para iPad/Proyector.
- **Core:** Motor de colisión vectorial secuencial (V17) + Dopamine Bloom.
- **Objetivo:** Refuerzo positivo y aprendizaje de trazos.
